# Thread

En este video-code se presenta el concepto de THREAD o HILO, para crear una tarea que se ejecuta fuera del programa principal.

Material requerido

Raspberry Pi Pico
Breadboard
LED (cualquier color)
Resistencia de 330 Ohms
Cables
